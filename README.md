# ⌨️ Typing Speed Test App

A simple and interactive typing speed test built using **pure JavaScript, HTML, and CSS**.  
This project is a solution to a **Frontend Mentor challenge**, focused on improving DOM manipulation, timing logic, and user interaction skills.

---

## 🚀 Features

- ⏱️ Real-time typing speed calculation (WPM - Words Per Minute)
- 🎯 Accuracy tracking
- 🔄 Random text generation for typing practice
- 🧠 Live input comparison with target text
- ⏳ Timer countdown or stopwatch mode
- 🔁 Restart test functionality
- 📱 Fully responsive design

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- DOM Manipulation
- setInterval / setTimeout for timing logic
